## Tableau de correspondance des broches (Hack LMN-3)

| Pin d'origine | Pin modifiée |
| --- | --- |
| 0 | 33 |
| 1 | 37 |
| 14 | 38 |

## Instruction de pliage

- Plier les broches **0**, **1** et **14** (les isoler en l'air).
- Relier ensuite ces signaux via fils volants vers **33**, **37** et **38** selon le tableau.
